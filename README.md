# nimbus

bsky client written in [nim]

```
nimble run
```

for MacOS users:
```
nimble build ; ./nimbus
```
---

### Setup Credentials

```
cp .env.example .env
```
```.env
PDSHOST=https://bsky.social
BLUESKY_HANDLE=foo.bar
APP_PASSWORD=letmein
```

> ref: https://bsky.app/settings/app-passwords
